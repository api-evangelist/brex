# Brex GraphQL API

Brex is a financial stack for startups and enterprises covering corporate cards, business accounts, bill pay, and expense management. The API covers accounts, cards, transactions, statements, expenses, vendors, and team management.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Brex REST API
**Documentation:** https://developer.brex.com/

- Reference: https://developer.brex.com/openapi/budgets_api/
- Schema: https://github.com/api-evangelist/brex
